# Unreal Engine Battery Man Game

**Unreal Engine Battery Man Game** is a simple 3D game developed using Unreal Engine where the player controls a third-person character and must collect batteries to survive. The player's health continuously decreases over time, and collecting batteries restores health. The game ends when the health bar reaches zero.
